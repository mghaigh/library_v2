
# Operation: top genres


Notes: Get the **Top Genre** filtered by the relevant parameters.
For the specified number of towns, the number of loans for each Genre is returned.


#### Parameters


| Name | Type | Notes | Required |
| :--- | :--- | :--- | :--- |
| townLimit | [Int](../../core-types/primitives/int.md) | The **townLimit** query parameter. |  |

#### Return Type


| Type | Notes |
| :---: | :--- |
| [TopGenre](interface-types/top-genre/top-genre.md) | The **Top Genre** returned by this operation. |
